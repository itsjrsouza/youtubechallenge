# Desafio: Clonar a página do Youtube com HTML e CSS

Este projeto marca o início da **Trilha de CSS da DIO** — um marco especial onde teoria encontra prática. Aqui, você vai transformar conceitos fundamentais em uma experiência visual imersiva.

[Clique aqui](https://itsjrsouza.github.io/youtubechallenge/) para acessar o resultado final da Landing Page criada a partir do desafio!

![image](https://github.com/itsjrsouza/youtubechallenge/blob/7854bf773d19c8f8ec803e3d208f0b8b9ad6fd7a/assets/Youtube.png)

### 🎯 O que você vai praticar

| Conceito | Aplicação |
|:---------|:----------|
| `Fundamentos CSS` | Seletores, especificidade e cascata |
| `Unidades de Medida` | Relativas (rem, %) vs Absolutas (px) |
| `Box Model` | Margin, padding, border e content |
| `Background-clip` | Textos com gradiente animado |
| `Layout Responsivo` | Flexbox e posicionamento |

[Link do Figma](https://www.figma.com/design/lrRWUZPKnqMDZrSDJmZxUS/Desafio-de-Flexbox---DIO?node-id=1-8&t=pUbiRRHwqVwmdzhq-0) contendo o protótipo do desafio para
que você possa se basear.

Caso tenha alguma dúvida, ou queira comparar o resultado do desafio que você fez, nós temos o site finalizado na branch *final*. Basta alterar a branch do projeto
utilizando o comando `git checkout final` no seu terminal.
